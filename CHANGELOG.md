# Changelog

## 0.1.1 - 2026-02-11

- README quickstart now uses the real public repo URL.
- Added explicit guidance on not using tokens in URLs/commands.
- Token file permissions are auto-hardened to owner-only on POSIX systems.
- Documented CI secret scanning protections.

## 0.1.0 - 2026-02-11

- Initial public release.
- Added `auth` command for OAuth token creation.
- Added `sync` command to import/update/delete Meetup events from Gmail ICS invites.
- Added macOS launchd installer script for daily scheduling.
